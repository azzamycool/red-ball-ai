# Red Ball AI 🔴

An endless platformer game featuring a Red Ball, built with HTML5 Canvas and an AI-driven level generator.

## Features
- **Infinite Levels:** The AI generates unique paths, platforms, and enemies every time you progress.
- **Physics-Based Movement:** Smooth squash and stretch animations.
- **Dynamic Environments:** Multiple themes including Forest, Desert, and Space.
- **Mobile Friendly:** Includes touch controls for mobile browsers.

## How to Play
1. Open `index.html` in any modern web browser.
2. Use **Arrow Keys** or **WASD** to move and jump.
3. Collect coins and reach the flag to proceed to the next AI-generated level.
4. Avoid spikes and enemies!
